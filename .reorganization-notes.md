# YourFreeDownloader

Este es un proyecto multiplataforma para descargar videos y audio de YouTube.

## Estructura del Proyecto

- `mobile-android/` - Aplicación Android
- `desktop-multiplatform/` - Aplicación de escritorio para Windows y Linux
- `shared/` - Código compartido (futuro)

## Carpetas Antiguas (Puedes eliminarlas)

Las siguientes carpetas contienen código antiguo y pueden ser eliminadas de forma segura:
- `app/` - Proyecto Android antiguo (ahora en mobile-android/)
- `YourFreeDownloader-Windows/` - Proyecto Windows antiguo (ahora en desktop-multiplatform/)
- `build/` - Archivos de compilación antiguos
